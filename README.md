# machine-learning-project
Repositori ini berisi proyek analisis data dan machine learning untuk mengeksplorasi faktor-faktor yang memengaruhi harapan hidup (life expectancy) suatu negara menggunakan dataset dari WHO. Proyek ini mencakup dua pendekatan utama: Klasifikasi (menentukan status negara: Developed vs Developing) dan Regresi (memprediksi nilai harapan hidup).
